# Release Notes

This directory contains detailed release notes for each GLOST version.

## Available Release Notes

- **[v0.5.0](./RELEASE_NOTES_v0.5.0.md)** - Unified-style processor and registry system
  - Major architectural update
  - New processor API with fluent interface
  - Enhanced plugin registry
  - CLI tools and preset system
  - 244 tests passing, production-ready

## Release Note Format

Each release note includes:
- **Overview** - What's new in the release
- **New Features** - Detailed feature descriptions with examples
- **Breaking Changes** - What needs to be updated
- **Migration** - Links to migration guides
- **Performance** - Benchmark results
- **Documentation** - Links to guides and references

## Version Numbering

GLOST follows [Semantic Versioning](https://semver.org/):
- **Major** (v1.0.0) - Breaking changes
- **Minor** (v0.5.0) - New features, backwards compatible
- **Patch** (v0.5.1) - Bug fixes
