# glost-presets

Pre-configured plugin combinations for common use cases.

## Installation

```bash
npm install glost-presets glost-core glost-processor
```

Or use the main `glost` package which includes presets:

```bash
npm install glost
```

## Features

- Ready-to-use plugin combinations
- Optimized for specific use cases
- Easy to extend and customize
- TypeScript support

## Quick Start

```typescript
import { glost } from 'glost';
import { languageLearningPreset } from 'glost/presets';

const processor = glost()
  .use(languageLearningPreset)
  .freeze();

const result = await processor.process(document);
```

## Available Presets

### languageLearningPreset

Comprehensive preset for language learning applications.

**Includes:**
- Frequency analysis
- Difficulty scoring
- Part-of-speech tagging
- Transcription support

```typescript

  .use(languageLearningPreset)
```

- Vocabulary learning apps
- Flashcard systems

## Extending Presets

Add additional plugins to a preset:

```typescript
import { glost } from 'glost';
import { languageLearningPreset } from 'glost/presets';
import { myCustomPlugin } from './my-plugin';

const processor = glost()
  .use(languageLearningPreset)
  .use(myCustomPlugin)  // Add custom plugin
  .freeze();
```

## Creating Custom Presets

Define your own preset:

```typescript
import type { Preset } from 'glost-presets';
import { createFrequencyExtension } from 'glost-frequency';
import { createPosExtension } from 'glost-pos';

export const myCustomPreset: Preset = {
  name: 'my-custom-preset',
  description: 'Custom preset for my app',
  plugins: [
    ...createFrequencyExtension({ ... }),
    ...createPosExtension({ ... })
  ]
};

// Use it
const processor = glost()
  .use(myCustomPreset)
  .freeze();
```

## Preset with Options

Some presets accept configuration:

```typescript
import { createLanguageLearningPreset } from 'glost/presets';

const customPreset = createLanguageLearningPreset({
  targetLanguage: 'th',
  includeFrequency: true,
  includeDifficulty: true,
  includePOS: false  // Disable POS tagging
});

const processor = glost()
  .use(customPreset)
  .freeze();
```
