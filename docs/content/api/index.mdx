# API Reference

Complete API documentation for `glost-common`.

## Modules

- [Language Codes](./language-codes) - Convert and validate language codes
- [BCP-47](./bcp-47) - Parse and build BCP-47 tags
- [Proficiency](./proficiency) - Work with proficiency levels
- [Conversions](./conversions) - Language-specific proficiency conversions
- [Constants](./constants) - Data and schemas

## Quick Examples

```typescript
import {
  toISO639_3,
  parseBCP47,
  cefrToNumeric,
  jlptToNumeric
} from 'glost-common'

// Language codes
toISO639_3("en")       // "eng"
toISO639_3("zh-Hans")  // "zho"

// BCP-47 parsing
parseBCP47("zh-Hans-CN")
// { language: "zh", script: "Hans", region: "CN" }

// Proficiency
cefrToNumeric("B1")    // 3
jlptToNumeric("N3")    // 3
```
