# Quick Start Example

This example demonstrates the basic usage of GLOST for creating and manipulating glossed syntax trees.

## What's Covered

- Creating simple GLOST documents
- Working with words, sentences, and paragraphs
- Basic multilingual support
- Tree traversal and manipulation

## Running the Example

```bash
pnpm test
```

## Files

- `src/__tests__/basic-usage.test.ts` - Basic GLOST operations
- `src/__tests__/multilingual.test.ts` - Multilingual document handling
