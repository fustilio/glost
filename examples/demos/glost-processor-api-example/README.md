# Processor API Example

Examples demonstrating the unified-style processor API for building language learning applications and processing pipelines.

## What's Covered

- Using the `.use()` fluent API
- Applying presets for common use cases
- Building language learning readers
- Processing documents with middleware hooks

## Running the Example

```bash
pnpm test
```

## Files

- `src/__tests__/language-learning-reader.test.ts` - Building a language learning reader
- `src/__tests__/language-learning-app.test.ts` - Complete language learning application
