# Extensions API Example

Examples demonstrating how to use and create GLOST extensions for AST transformations and metadata enrichment.

## What's Covered

- Composition patterns for chaining extensions
- Creating custom extensions
- Using transcription extensions
- Extension lifecycle and processing

## Running the Example

```bash
pnpm test
```

## Files

- `src/__tests__/composition-pattern.test.ts` - Composing multiple extensions
- `src/__tests__/custom-extension.test.ts` - Creating custom extensions
- `src/__tests__/transcription.test.ts` - Transcription extension usage
- `src/__tests__/using-extensions.test.ts` - Extension system basics
